# Codex Instructions

You are contributing to a premium personal brand website for a French-speaking influencer.

This is **not** an e-commerce website.

The products exist to support the brand.

The person is the product.

---

# Objective

Build a premium, modern, trustworthy website that makes visitors immediately think:

> "I want to follow this person."

Every design decision should strengthen the creator's personal brand.

The website should feel closer to Apple, Nike or Notion than a typical MLM or Shopify template.

---

# Target Audience

French-speaking audience.

People interested in:

- Wellness
- Personal development
- Entrepreneurship
- Healthy lifestyle
- Motivation

---

# Component Rules

- One responsibility per component.
- Keep components small.
- Reusable whenever possible.
- Avoid giant files.
- Avoid unnecessary abstractions.
- Functional components only.

# Accessibility

Semantic HTML.

Keyboard friendly.

# Performance

Prefer server components whenever possible.

Avoid unnecessary client components.

Avoid unnecessary JavaScript.

Keep DOM clean.

---

# Code Style

Readable.

Senior-level.

Self-explanatory.

Avoid comments explaining obvious code.

Extract repeated UI into reusable components.

---

# Workflow

For every ticket:

1. Understand the requested section.

2. Explain the implementation plan.

3. Implement.

4. Keep the existing design language.

Never redesign previous sections unless explicitly requested.
